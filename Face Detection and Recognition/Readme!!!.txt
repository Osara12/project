Face Detection and Recognition
1.Create folder "faceRecognized" in C:/ as Database
2.When register a new face in program.You need to add profile picture in "C:/faceRecognized" with the same name that you register.
3.This program need ".Net framwork 3.5"

-------------------------------------------------------------------------------------
1.สร้างแฟ้มชื่อ "faceRecognized" ใน C:/ เพื่อเป็นฐานข้อมูล
2.เมื่อลงทะเบียนใบหน้าใหม่ในโปรแกรม ต้องเพิ่มรูปในฐานข้อมูล C:/faceRecognized และตั้งชื่อให้เหมือนที่ลงทะเบียนไว้
3.ต้องการ ".Net framwork 3.5" ในการทำงาน
